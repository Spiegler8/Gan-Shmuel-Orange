Welcome

This is the Weight System of Gan Shmuel 🌳

Here are the available endpoints:

  • POST /weight: Add new weight session
  • GET /weight: View weight sessions
  • GET /item: Lookup truck or container by ID
  • GET /session: Lookup session by ID
  • GET /unknown: Find unknown containers
  • POST /batch-weight: Upload container weights (CSV/JSON)
Use the tabs above to access each API endpoint form.
